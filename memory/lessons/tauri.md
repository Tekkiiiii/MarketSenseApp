## Patterns

- Window config in `tauri.conf.json`: default size, min size, app title, bundle ID
- Rust backend in `src-tauri/src/` with Tauri commands
- `npm run tauri dev` for development, `npm run tauri build` for production
- node-pty for terminal emulation inside Tauri
