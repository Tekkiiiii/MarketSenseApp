# Changelog

All notable changes to **ThuongMinh** will be documented in this file.

## [1.0.0] — 2026-04-09

### Added
- Initial public release: **macOS .dmg** (Apple Silicon + Intel) + **Windows .exe** (NSIS installer)
- Landing page at [tintai](https://tintai.pages.dev)
- Vietnamese financial news RSS monitoring via Ollama (Gemma 4)
- Bullish / Bearish / Neutral AI verdicts with Vietnamese summaries
- 100% local processing — no data leaves the device
- Desktop-first responsive layout (1280×800, resizable down to 320×568)
